Néhány fontos adat:

A számitógépen, melyen a program készült a következõ szoftverek futottak:

	-os: 			WIN10 x64
	-forráskódszerkesztõ:	Netbeans 8.2

1.Lépés
	Töltsd le a fájlt. 

2.Lépés
	Csomagold ki a zip-et.

3.Lépés
	Indítás: A futtatáshoz szükséges, hogy telepítve legyen a legfrissebb JAVA JRE .
		-közvetlen indítás:  \dist\SpaceBird.jar
		-közvetett indítás:  NetBeans vagy Eclipse-t haszálna projectként megnyitva a mappát és azon belül inditás.

	Szerkesztés:  NetBeans vagy Eclipse-t használva, illetve szükséges, hogy telepítve legyen a legfrissebb JAVA JDK. 
		
Megjegyzés: A Dong Nguyen által készített "Flappy Bird" játék inspirál.
